Welcome to the repository of my personal portfolio website, where engineering meets creativity. This project is built using **Astro** for high-performance static site generation and **Three.js** for immersive interactive 3D elements.

## ✨ Features

- 🏎️ **Optimized with Astro**: Fast, lightweight, and modern web architecture.
- 🌀 **Interactive 3D**: Specialized components using Three.js for visual engagement.
- 📱 **Fully Responsive**: Designed to look great on any device, from desktop to mobile.
- 🏗️ **Clean Architecture**: Built as a foundation for scalable, modern web applications.
- 🎨 **Minimalist Design**: Focused on developer experience and visual clarity.

## 🛠️ Tech Stack

*   **Framework:** [Astro](https://astro.build/)
*   **3D Engine:** [Three.js](https://threejs.org/)
*   **Styling:** Vanilla CSS & Modern Layouts
*   **Language:** TypeScript
*   **Deployment:** GitHub Pages (via GitHub Actions)

## 🚀 Getting Started

To run this project locally, follow these simple steps:

### 1. Clone the repository
```bash
git clone https://github.com/pmalirz/pmalirz.github.io.git
cd pmalirz.github.io
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the dev server
```bash
npm run dev
```

Your site should now be running at `http://localhost:4321/`!

## 🚢 Deployment

The website is automatically deployed to **GitHub Pages** whenever changes are pushed to the `main` or `master` branch.

- **Workflow:** `.github/workflows/deploy.yml`
- **Trigger:** Push to `main`/`master`
- **URL:** [https://pmalirz.github.io/](https://pmalirz.github.io/)
